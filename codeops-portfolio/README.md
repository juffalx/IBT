# Running the Code

Clone the repository:

```bash
git clone https://github.com/juffalx/IBT.git
```

Move into the project:

```bash
cd IBT/codeops-portfolio/Module-01
```

Open any day:

```bash
cd day05
```

Run:

```bash
python main.py
```

or

```bash
python practice.py
```

or

```bash
python project.py
```
